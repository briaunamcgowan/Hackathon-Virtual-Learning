HTML Intro

HTML headings are titles or subtitles that you want to display on a webpage. There is a max of 6 headings that can be used within HTML, all content used in between those headings must have an opening <> and closing </> symbol to complete the code. Attached below is a reference to a multitude of HTML tags commonly used within the heading tags. HTML Element Reference









Paragraphs

The HTML <p> element represents a paragraph. Paragraphs are considered to be a block of text composed of one or more sentences that refer to a particular topic. However, HTML paragraphs can be any structural grouping of related content, such as images or form fields. A paragraph always starts on a new line and browsers automatically add some white space (a margin) before and after a paragraph. HTML Paragraph Reference
